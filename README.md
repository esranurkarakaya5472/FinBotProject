# 💎 FinBot Pro: Yapay Zeka Destekli Finansal Analiz Asistanı

Bu proje, karmaşık finansal raporları (PDF) analiz etmek ve kullanıcıların bu raporlar üzerinden anlamlı veriler elde etmesini sağlamak amacıyla geliştirilmiş bir **AI Agent** uygulamasıdır.

## 🚀 Canlı Uygulama
Uygulamaya şu linkten ulaşabilirsiniz: 
[FinBot Pro Canlı Link](https://finbotproject-rdm3basrxyvhqrrppngfms.streamlit.app/)

## 🛡️ Siber Güvenlik ve Veri Gizliliği
Bir **Bilişim Güvenliği Teknolojisi** öğrencisi olarak projede şu güvenlik standartları uygulanmıştır:
- **Secrets Management:** Gemini API anahtarları kod içerisine gömülmemiş, Streamlit Cloud üzerinde güvenli 'Secrets' katmanında maskelenmiştir.
- **Environment Isolation:** Hassas verilerin sızmasını önlemek amacıyla `.gitignore` yapılandırması kullanılmıştır.
- **Data Privacy:** Yüklenen veriler sadece oturum bazlı işlenmekte ve sunucuda kalıcı olarak depolanmamaktadır.

## 🛠️ Kullanılan Teknolojiler
- **Dil:** Python 3.12
- **Model:** Google Gemini 1.5 Flash
- **Arayüz:** Streamlit (Ultra-Vibrant Glow Theme)
- **Kütüphaneler:** PyPDF2, LangChain, Google Generative AI
