💎 FinBot Pro: AI-Powered Financial Intelligence
FinBot Pro bir "Chatbot" değildir; döküman tabanlı bir "Finansal Karar Destek Sistemi"dir.

🎯 Temel Vizyon
Zekâ Katmanı: Sadece döküman verisiyle konuşan, "halüsinasyon" üretmeyen güvenli LLM mimarisi.

Hız & Verimlilik: Yüzlerce sayfalık finansal raporu saniyeler içinde analiz eden RAG (Retrieval-Augmented Generation) altyapısı.

Güvenlik: API anahtarları ve verilerin maskelendiği, Bilişim Güvenliği standartlarında modüler kod yapısı.

🛠️ Teknik Mimari
Proje, sürdürülebilirlik için 4 ana modüle ayrılmıştır:

llm_client.py: Gemini 1.5 Flash entegrasyonu.

embedding.py: PDF'den vektörel veri dönüşümü.

retriever.py: Akıllı bilgi geri çağırma (RAG).

app.py: Streamlit kullanıcı arayüzü.

## 🛠️ Kullanılan Teknolojiler
- **Dil:** Python 3.12
- **Model:** Google Gemini 1.5 Flash
- **Arayüz:** Streamlit (Ultra-Vibrant Glow Theme)
- **Kütüphaneler:** PyPDF2, LangChain, Google Generative AI
